# Cookie Run: Ovenbreak To-Do Specs

This folder is the source of truth for the project. The website helps Cookie Run: Ovenbreak players build leveling to-do lists for cookies, pets, and treasures.

## Core Specs

- `global-spec.md` - product goals, audience, MVP scope, and success criteria
- `rules.md` - universal rules, level caps, terminology, and assumptions
- `functional-map.md` - pages, user flows, and feature areas
- `data-model.md` - entities, relationships, list formats, and catalog data
- `ui-spec.md` - layout, interaction, and catalog/list UI guidance

## Feature Specs

- `features/001-catalog-pages.md` - cookies, pets, treasures, sorting, and cookie-pet pairing prompt
- `features/002-list-formats.md` - the five supported to-do list formats
- `features/003-preset-lists.md` - Trophy Race, Breakout, Champions League, and Guild Run presets
- `features/004-custom-lists.md` - user-created named lists

## Template

- `default-spec.md` - reusable template for future feature specs

## Working Rule

If implementation changes behavior, update the relevant spec in the same task. If a requirement is ambiguous, add it to the relevant `Open Questions` section instead of inventing hidden behavior.
